# Strength-calculated
Strength calculated is a project built for those looking to both calculate and track there current strength levels  
Created using the MERN stack and hosted on heroku  

## Features  
Unique design  
Data visulization  
Strength calculations  

### Technology used  
 - React (Frontend)
 - Express (Backend)
 - D3
 - Bootstrap
 - Sass
